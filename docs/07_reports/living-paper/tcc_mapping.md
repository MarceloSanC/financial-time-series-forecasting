# Mapeamento Living Paper -> TCC (ABNT)

## Objetivo
Mapear o que esta no living paper para os capitulos do texto em LaTeX/ABNT.

Detalhamento do objetivo:
- Garantir rastreabilidade entre rascunho tecnico e texto final do TCC.
- Reduzir perda de conteudo na transposicao do markdown para LaTeX.
- Assegurar consistencia entre problema, metodo, resultados e conclusoes ao longo dos capitulos.
- Servir como checklist de sincronizacao para cada ciclo de escrita.

## Tabela de mapeamento (rascunho)
- `00_outline.md` -> Introducao (tema/problema/justificativa/objetivos/recorte)
- `10_problem_and_related_work.md` -> Revisao Teorica
- `20_method.md` -> Metodo
- `30_results_and_analysis.md` -> Resultados
- `40_limitations_and_conclusion.md` -> Conclusoes

## Observacoes
- Este arquivo sera refinado junto com a escrita dos capitulos em `text/`.
- Status atual: Introducao (`text/2_Introducao/2_Introducao.tex`) preenchida em v1 para secoes de contextualizacao, problema, justificativa, delimitacao e estrutura.
- Sincronizacao: resumo da Introducao v1 registrado em `00_outline.md` (secao 11).
