# Global Checklist

## Pré-escrita
- Ler `WRITING_PROTOCOL.md`.
- Confirmar tese, pergunta e recorte no `00_outline.md`.
- Confirmar seção-alvo no `chapter_structure_freeze.md`.

## Durante a escrita
- Manter tom acadêmico objetivo.
- Usar citações somente quando sustentarem decisão metodológica ou argumento central.
- Evitar claims absolutos.

## Pós-escrita
- Validar coerência com objetivos do trabalho.
- Validar consistência terminológica (TFT, OOS, quantis, leakage, reprodutibilidade).
- Sincronizar `tcc_mapping.md` e `evidence_log.md` quando aplicável.
