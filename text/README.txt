Template para Trabalho de Conclusão de Curso da Universidade Federal de Santa Catarina (UFSC)

- As configurações devem ser realizadas no arquivo Main.tex;

- O conteúdo do trabalho deve ser inserida nos arquivos de cada pasta;

- Para gerar o arquivo em pdf compile o arquive Main.tex;

- A lista de siglas deve ser compilada separadamente no terminal, para isso utilize o seguinte comando $makeindex ARQUIVO_PRINCIPAL.nlo -s nomencl.ist -o ARQUIVO_PRINCIPAL.nls

Obs.: Substitua o ARQUIVO_PRINCIPAL pelo nome do arquivo;

- Em caso de não utilização da folha de aprovação, listas, apêndices ou anexo exclua o comando correspondente no arquivo Main.tex;

- Atenção ao inserir as referências no Jabref! O cadastramento errado pode gerar uma referência errada;

- Para consultar a documentação de referências digite no terminal: $texdoc
abntex2cite
